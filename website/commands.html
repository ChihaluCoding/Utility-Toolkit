コマンドリファレンス

建築支援向けの専用コマンドをまとめています。プリセット呼び出しや履歴参照をチャットから素早く実行。

Download

主要コマンド

/busp preset

  • `/busp preset save <名前>`: 現在のホットバーを保存。
  • `/busp preset load <名前>`: 保存済みプリセットを呼び出し。
  • `/busp preset list`: 登録一覧をチャットに出力。

/busp village

  • 村の構造物を検索して座標を一覧化。
  • `/busp village next`: 最寄りを自動追尾。
  • 経路の始点と終点をビーコン粒子で表示。

/busp history

  • 使用履歴タブと同じデータをチャットで閲覧。
  • `/busp history clear`: 全ワールド／ワールド別でリセット可能。
  • `/busp history pin `: 履歴アイテムをお気に入りへ固定。

便利なショートハンド

テンプレ管理

  • `/busp template save house-01` で構造登録。
  • `/busp template list --detail` で材料統計も表示。
  • `/busp template build house-01` で即配置を呼び出し。

環境設定

  • `/busp config reload` でJSONを再読み込み。
  • `/busp config set favorites.limit 40` のようにCLI設定。
  • 設定はGUIと同期され、即時反映されます。

デバッグ

  • `/busp debug overlay on` でヒントラインを表示。
  • `/busp debug log` で最新の自動化ログを取得。
  • 問題が起きた場合はこのログを添えて報告してください。

権限とセキュリティ

Permission

  • サーバーでは `building-support.admin` 権限が必要。
  • PresetやHistoryのみを許可したい場合は細分化されたノードを利用。
  • 危険コマンドはコンフィグで完全に無効化できます。
